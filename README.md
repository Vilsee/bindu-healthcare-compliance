# Bindu Healthcare Compliance Agent & Dashboard

This repository contains the full implementation of the Healthcare Compliance Agent and its associated high-tech landing page dashboard.

## Structure
- `/agent`: The backend microservice integrated with `bindufy`.
- `/frontend`: The premium industrial-themed landing page.

## Getting Started

### Backend Agent
1. Navigate to `/agent`
2. Run `python agent.py`

### Frontend Dashboard
1. Navigate to `/frontend`
2. Open `index.html` in your browser.

## Project Vision
To provide a compliance-first, evidence-linked clinical decision support system built on the Bindu Internet of Agents platform.
